# RETECH ELECTRICAL – Business Website

A full‑stack web application for an electrical services business. Features include:

- Dynamic service cards with admin‑uploadable images
- Home page background and About Us image management
- Team member management (add, edit, delete with photos)
- Customer appointment booking
- WhatsApp & Facebook links configurable via admin panel
- Hidden admin login (triple‑click footer)
- All data persisted in a JSON file

## Tech Stack

- **Backend:** Node.js, Express, Multer (file uploads), express‑session
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Storage:** File‑based (data.json) + uploads folder
